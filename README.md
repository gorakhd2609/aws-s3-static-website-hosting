# aws-s3-static-website-hosting
## Project Overview

This project demonstrates hosting a static website on Amazon S3 using AWS Static Website Hosting.

The website was developed using HTML and CSS and deployed to an S3 bucket configured for web hosting.

## AWS Services Used

- Amazon S3
- Bucket Policies
- Static Website Hosting
- IAM (Basic Access Management)

## Features

- Static Website Deployment
- Bucket Configuration
- Object Permissions
- Website Hosting Configuration
- Basic Cloud Security Concepts

## Deployment Steps

1. Created an S3 bucket.
2. Uploaded website files.
3. Enabled Static Website Hosting.
4. Configured bucket permissions.
5. Applied bucket policy.
6. Tested website accessibility.

## Key Learnings

- Amazon S3 Fundamentals
- Static Website Hosting
- Access Management
- Bucket Policies
- Cloud Storage Concepts
